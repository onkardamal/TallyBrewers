# Security Rules

Passwords are never stored.

Private Keys never leave user device.

Only Public Keys are stored.

HTTPS Required.

Secure Cookies.

HttpOnly Cookies.

SameSite Cookies.

JWT Rotation.

Refresh Token Rotation.

Rate Limiting.

CSRF Protection.

SQL Injection Protection.

XSS Protection.

Audit Logging.

Recovery Codes stored as hashes.
